Sobre o Área Salva

O Área Salva é um aplicativo simples de gerenciamento da área de transferência para dispositivos Android. Ele permite que os usuários coletem e organizem textos copiados para fácil acesso posterior.

Funcionalidades

  • Monitoramento da Área de Transferência: O aplicativo monitora a área de transferência do dispositivo para detectar textos copiados.
  • Salvar Textos: Os textos copiados são salvos automaticamente no aplicativo para referência futura.
  • Excluir Textos: Os usuários podem excluir itens da lista de textos salvos, mantendo a área de transferência limpa e organizada.
  • Interface Intuitiva: Uma interface de usuário simples e intuitiva permite uma navegação fácil e rápida.

Requisitos

  • Android: 6.0 (Marshmallow) ou superior
  • Permissões: Acesso à leitura e escrita da Área de Transferência

Histórico de Versões

Versão 2.1

  • Repaginação da Tela de Início: A tela inicial do aplicativo foi totalmente renovada, oferecendo uma interface mais moderna e intuitiva para melhorar a experiência do usuário.
  • Correções Ortográficas: Foram realizadas correções na ortografia e gramática da língua portuguesa em todo o aplicativo, garantindo uma comunicação mais clara e correta.
  • Autenticação Biométrica ou Palavra-Chave: Para acessar o aplicativo, é necessária autenticação biométrica ou uma palavra-chave, aumentando a proteção dos seus dados.
  • Barreira Anti-Captura de Tela: Implementada uma barreira que impede a captura de tela do aplicativo, garantindo a privacidade das informações.
