Sobre o ShareXpress

ShareXpress é um aplicativo Android que permite a transferência de arquivos entre dispositivos Android e PCs sem a necessidade de MTP (Media Transfer Protocol). Esta versão para Android é um port da aplicação original desenvolvida em Python utilizando Qt5, oferecendo uma alternativa eficiente para o compartilhamento de arquivos.

Funcionalidades

  • Transferência de Arquivos: Envie arquivos do seu dispositivo Android para um PC e vice-versa.
  • Interface Amigável: Visualize o progresso da transferência de arquivos com barras de progresso em tempo real.
  • Gerenciamento de Arquivos: O aplicativo permite a seleção e envio de múltiplos arquivos, bem como a visualização dos arquivos recebidos.
  • Verificação de Espaço: O aplicativo verifica o espaço disponível no dispositivo antes de iniciar a transferência.

Requisitos

  • Android: 6.0 (Marshmallow) ou superior
  • Permissões: Acesso à leitura e escrita no armazenamento externo
  • Linux: Fedora Linux 38 ou superior

Histórico de Versões

Versão 1.8

  • Implementação da funcionalidade de descoberta de servidores disponíveis na rede para Android e Python.
  • Correção de problemas de timeout e melhorias na comunicação entre o cliente Python e o servidor Kotlin.
  • Ajustes na interface do usuário para exibição correta dos servidores encontrados.

Versão 1.7

  • Compartilhamento de Conteúdo da Área de Transferência: A versão de PC do ShareXpress agora suporta o compartilhamento de conteúdo da área de transferência. Você pode enviar o texto da área de transferência diretamente do seu PC para o servidor, assim como pode fazer com os arquivos.
  • Melhoria na Interface: A interface de usuário foi otimizada para oferecer uma experiência mais intuitiva e fluida.
  • Correção de Bugs: Diversos bugs foram corrigidos para melhorar a estabilidade e o desempenho do aplicativo.
  • Versões Disponíveis: O ShareXpress está disponível para Android e PC. Use a versão de PC para uma integração mais robusta e funcionalidades adicionais em seu ambiente de desktop.

Versão 1.6

  • Suporte a arquivos grandes: Implementado suporte para envio de arquivos maiores que 1GB. Agora, o aplicativo consegue processar e transferir grandes volumes de dados sem problemas.
  • Nota: A versão mobile já está disponível, oferecendo a mesma robustez e confiabilidade no envio de arquivos grandes. Atualize para aproveitar essas melhorias e ter uma experiência mais estável e confiável com o aplicativo. Para mais detalhes e download, acesse ShareXpress Android.

Versão 1.5

  • Exibição Aprimorada do Progresso: Porcentagem de Progresso: Agora é possível visualizar a porcentagem de progresso de cada arquivo durante a transferência, fornecendo uma visão mais clara do andamento. Tempo Estimado: Adicionada a exibição do tempo estimado restante para cada transferência de arquivo, ajudando o usuário a saber quanto tempo falta para concluir o processo.
  • Atualizações Suaves da Interface: Implementação de técnicas para garantir que operações pesadas sejam realizadas em segundo plano, permitindo que a interface do usuário seja atualizada de maneira mais fluida e eficiente.
  • Operações de Arquivo: Remoção de Arquivo: Agora é possível remover um arquivo da lista de transferência com um clique direito (ou ação semelhante), com a funcionalidade de remoção desativada durante as transferências ativas para evitar interrupções.
  • Alertas de Status de Conexão: Inclusão de diálogos de alerta (AlertDialog) para informar o usuário sobre o status da conexão, falta de seleção de arquivos ou ausência de IP e porta.
  • Correção de Travamento: Resolvido o problema de travamento da interface do usuário durante as transferências de arquivos, garantindo que o aplicativo continue responsivo.
  • Gerenciamento de Permissões: Ajustadas as permissões necessárias para garantir que o aplicativo funcione corretamente em versões recentes do sistema operacional.
  • Aprimoramentos de Desempenho: Melhorias no desempenho geral das transferências de arquivos, tornando o processo mais rápido e eficiente.
  • Atualizações Visuais: Melhorias visuais na interface do usuário para uma experiência mais intuitiva e agradável.

Versão 1.4

  • Adaptações no Cliente: O cliente foi modificado para se adaptar ao servidor do Android que está sendo desenvolvido em Kotlin, com melhor compatibilidade, ajustes na interface gráfica e melhorias na estabilidade e performance do cliente.
  • Desenvolvimento da Versão Android: O desenvolvimento da versão Android começou, trazendo a mesma facilidade e eficiência do ShareXpress para dispositivos Android.
  • Funcionalidades: Envio de arquivos entre dispositivos na mesma rede local, interface gráfica intuitiva, compatibilidade com diversos formatos de arquivos e adaptação automática ao servidor Android.

Versão 1.3

  • Envio de Arquivos: Transfira arquivos de um dispositivo para outro de forma rápida e segura.
  • Recebimento de Arquivos: Receba arquivos enviados por outros dispositivos com facilidade.
  • Interface Gráfica Amigável: Uma interface intuitiva e fácil de usar, desenvolvida com PyQt5, garantindo uma experiência de usuário agradável.
  • Propósito e Visão: Simplificar o processo de compartilhamento de arquivos, eliminando a necessidade de soluções complexas ou dependência de serviços de terceiros.

  • Próximos Passos: Comprometidos em melhorar continuamente o ShareXpress e adicionar novas funcionalidades com base no feedback dos usuários, com planos de lançar a versão para Android.
  • Agradecimentos: Agradecimentos a todos que contribuíram para o desenvolvimento e teste desta primeira versão.
